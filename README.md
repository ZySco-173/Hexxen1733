HeXXen 1733 Foundry VTT Macros
Dieses Repository enthält eine Sammlung von Makros für HeXXen 1733, die speziell für die Nutzung in Foundry Virtual Tabletop entwickelt wurden. Die Skripte unterstützen Spielleiter und Spieler dabei, Spielmechaniken effizienter zu gestalten und den Spielfluss zu beschleunigen.

Technische Spezifikationen
Um eine reibungslose Funktionalität zu gewährleisten, wurden die Makros unter folgenden Bedingungen erfolgreich getestet:

Foundry VTT Version: 13 (Stable)

HeXXen 1733 System-Version: 2.0.8

Abhängigkeiten: Es werden keine zusätzlichen Module (wie libWrapper oder Advanced Macros) benötigt. Die Makros funktionieren "Out-of-the-Box" mit der Standard-API von Foundry.

Installation
Öffne dein Foundry VTT Projekt.
Erstelle ein neues Makro in der Makroleiste.
Setze den Typ auf "Script".
Kopiere den Inhalt der gewünschten .js Datei aus diesem Repository und füge ihn in das Textfeld ein.

Speichere das Makro ab.
